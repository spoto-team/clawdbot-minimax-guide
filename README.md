# Clawdbot MiniMax 配置指南

详细的 Clawdbot 国内版 MiniMax M2.1 API 配置文档，帮助用户快速配置和使用 MiniMax API。

## 项目简介

本仓库提供了 Clawdbot 与国内版 MiniMax M2.1 API 集成的完整配置指南。Clawdbot 是一个强大的 AI 智能体 CLI 工具，通过本指南，您可以轻松配置 MiniMax 的 M2.1 模型，享受高效的 AI 编程辅助体验。

## 主要特性

- 📚 详细的配置步骤说明
- 🔧 完整的配置文件示例
- 🐛 常见问题解决方案
- ⚙️ 高级配置选项
- 🔒 安全建议和最佳实践

## 快速开始

1. 克隆本仓库或直接查看 `clawdbot-minimax-config-guide.md` 文件
2. 按照文档中的步骤安装和配置 Clawdbot
3. 配置 MiniMax API Key 和相关设置
4. 验证配置并开始使用

## 系统要求

- Node.js ≥ 22
- macOS、Linux 或 Windows (WSL2) 系统
- MiniMax 国内版 API Key

## 支持的模型

- MiniMax M2.1 (200k 上下文窗口)
- MiniMax abab6.5s (及其他兼容模型)
